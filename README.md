# 🚀 AutoPost Hub

Social media auto-uploader — upload once, publish everywhere.

## Tech Stack
- **Frontend:** Next.js (React)
- **Backend:** Python FastAPI
- **Queue:** Celery + Redis
- **Database:** PostgreSQL

## Supported Platforms
YouTube, Facebook, Instagram, TikTok, X (Twitter), Threads

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend (Coming Soon)
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
